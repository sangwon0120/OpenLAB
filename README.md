# OpenLAB

OpenLAB은 연구를 더 가볍게 시작할 수 있게 돕는 플랫폼을 소개하는 간단한 랜딩 페이지입니다. 학부생과 연구실을 연결해, 큰 프로젝트를 작은 업무로 나누어 부담은 낮추고 참여는 쉽게 만듭니다.

## 한눈에 보는 웹페이지
- **핵심 메시지:** “Unbundling of Research”
- **핵심 가치:** Light Commitment, Heavy Impact
- **누구를 위한가요?** 연구 경험이 필요한 학생과, 간단한 업무 도움이 필요한 연구실

## 페이지 구성
- **Hero:** OpenLAB의 핵심 슬로건과 간단한 소개, 바로 시작할 수 있는 버튼
- **Problem & Solution:** 학생과 연구실이 겪는 고민을 짚고, 마이크로 프로젝트로 해결하는 방식 소개
- **How It Works:** 공고 → AI 스크리닝 → 인터뷰 → 참여로 이어지는 4단계 흐름
- **Featured Projects:** 실제로 있을 법한 예시 프로젝트 카드 3개
- **Business & Trust:** 인증서 기반 신뢰와 SaaS형 스크리닝 안내

## 이 페이지가 전하는 경험
- 복잡한 연구를 쉽게 시작할 수 있다는 자신감
- 짧고 명확한 참여 과정을 통한 가벼운 진입장벽
- 프로젝트를 한눈에 이해할 수 있는 깔끔한 카드형 정보
